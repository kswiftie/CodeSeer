How verification works

A program code is accepted as input and then an Abstract Syntax Tree is constructed from it.
Subtrees are then built from specific vertices such as FunctionDef, If, Loop, etc.
These subtrees are hashed, and the resulting sets of hashes are compared.

Results on Dataset

This verification method uses AST-based hashing and subtree comparison.

  • Recall: 0.7097
  • Precision: 0.8462
