Verification Class Overview

This verification class uses tokenization to find similarities between programs.

Types of Checks

1. Standard

Builds sorted sequences of tokens for each program. Calculates Levenshtein distance and the angle between them.

  • Accuracy: 0.532
  • Recall: 0.419
  • Precision: 0.542
  • F1 Score: 0.473
  • Count of pairs: 124
  • Time: 50.51 seconds

2. NN (Neural Network Based)

This method uses Unicode-based token sequences and generates embeddings. Then calculates angle and Euclidean distance between embeddings.

  • Accuracy: 0.355
  • Recall: 0.258
  • Precision: 0.320
  • F1 Score: 0.286
  • Count of pairs: 124
  • Time: 211.43 seconds
